Mengenal Semantic Element Di HTML5

  • Article

Ketika ingin membuat layout untuk menampilkan block artikel-artikel yang ada diwebsite kita, tag articel bisa kita gunakan untuk dengan element header dan footer Karena tag article memiliki type block, artinya elemen ini memiliki lebar sesuai lebar browser jika kita menggunakna lebarnya .

  • Aside

Elemen ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelasakan identitas penulis artikel. Karena Aside memiliki type / sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
